# Cost-Optimized End-to-End CI/CD Pipeline on AWS Free Tier

## Tools Used
- AWS EC2 (t2.micro)
- Jenkins
- Terraform
- Ansible
- GitHub

## Project Objective
Build a complete CI/CD pipeline using Jenkins that:
- Provisions infrastructure using Terraform
- Configures servers using Ansible
- Deploys a web application automatically
- Runs entirely on AWS Free Tier
